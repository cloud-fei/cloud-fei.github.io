Linux查找/查看一些信息

2023/6/27 linux

查找文件/服务

查找文件 find

-name：按文件名来查找文件
-prune：不在当前指定目录中查找

# 查找 / 目录下所有名称为hosts的文件（可使用 * 模糊匹配）
[root@node1 ~]# find / -name hosts
/etc/hosts

查找系统命令位置 which

[root@node1 ~]# which echo
/usr/bin/echo

查找命令是自带的还是外部安装的 type

[root@node1 ~]# type cd
cd is a shell builtin
[root@node1 ~]# type ssh
ssh is /usr/bin/ssh

查找端口号/服务是否开启 netstat 

# 查找端口号对应的服务
[root@node1 ~]# netstat -nlp | grep 22
tcp        0      0 0.0.0.0:22              0.0.0.0:*               LISTEN      1125/sshd
tcp6       0      0 :::22                   :::*                    LISTEN      1125/sshd
unix  2      [ ACC ]     STREAM     LISTENING     635      227/@usr/sbin/plymo  @/org/freedesktop/plymouthd

# 查找服务对应的端口号
[root@node1 ~]# netstat -nlp | grep ssh
tcp        0      0 0.0.0.0:22              0.0.0.0:*               LISTEN      1125/sshd
tcp6       0      0 :::22                   :::*                    LISTEN      1125/sshd

想使用yum下载rpm包而不安装(yumdownloader)

1、先安装yum-utils
yum install yum-utils -y

2、下载指定rpm包
yumdownloader tree

3、依赖一起下载加 --resolve

4、下载到指定目录加 --destdir=/opt/dir/
